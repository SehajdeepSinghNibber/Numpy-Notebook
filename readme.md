# 📊 NumPy Learning Repository

A beginner-friendly collection of **Python scripts** to learn and practice **NumPy**, the core library for numerical computing in Python.

---

## 🚀 About

This repository is designed to help you understand NumPy through **simple, hands-on Python scripts** instead of notebooks. Each file focuses on a specific concept, making it easy to learn step by step.

Whether you're starting with data science or strengthening your Python fundamentals, this repo will give you a solid base.

---

## 📂 Topics Covered

* ✅ Array creation
* ✅ Indexing and slicing
* ✅ Reshaping arrays
* ✅ Mathematical operations
* ✅ Vectorization
* ✅ Broadcasting
* ✅ Aggregate Functions
* ✅ Filtering
* ✅ Real-world examples

---

## 🛠️ Requirements

Make sure you have Python installed, then install NumPy:

```bash
pip install numpy
```
